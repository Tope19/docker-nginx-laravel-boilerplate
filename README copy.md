# Laravel 10 Application Boilerplate

This Laravel 10 application boilerplate is designed to kickstart your development process. It comes with an out-of-the-box Nginx configuration, Dockerized environments including PostgreSQL and RabbitMQ services, and a ready-to-go structured `Dockerfile` and `docker-compose.yaml` file.

## Prerequisites

Before you begin, ensure you have the following installed on your system:
- Docker Desktop
- Composer
- Git
